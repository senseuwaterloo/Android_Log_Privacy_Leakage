# Data Description

## APK Dataset

The original APK dataset consists of 48,702 Google Play applications from 2016-2021, hosted on AndroZoo.

### Access Information
- **Source**: [AndroZoo](https://androzoo.uni.lu/)
- **Size**: ~48,702 APKs
- **Time Period**: 2016-2021
- **Access**: Requires API key registration
- **Citation**: Please cite the AndroZoo paper when using their dataset

### Dataset Characteristics
- **Total Applications**: 48,702
- **Applications with Sources**: ~1,663 (3.4%)
- **Applications with Sinks**: ~1,663 (3.4%)
- **Leaking Applications**: 820 (49.3% of logging-enabled apps)

## Processed Results

### FlowDroid Output
- **Raw Findings**: ~19,000 initial leak detections
- **After Filtering**: 1,324 valid leaks (7% retention rate)
- **False Positive Rate**: 93% (filtered out)

### Analysis Results
- **Library Distribution**: Timber (35.2%), SLF4J (35.1%), Firebase (29.4%)
- **Data Flow Types**: Direct (37.5%), Indirect (62.5%)
- **Source Categories**: User-info (69.7%), User-input (30.3%)
- **Temporal Trend**: 68.2% of apps show improved privacy practices

## File Formats

### CSV Files
- **FlowDroid Output**: Contains source, sink, and path information
- **Analysis Results**: Structured data for each research question
- **Metadata**: App information from AndroZoo

### Configuration Files
- **Sink/Source Specification**: Text file with method signatures
- **Analysis Parameters**: YAML/JSON configuration files

## Data Availability

### Publicly Available
- **Processed Results**: [Figshare](https://figshare.com/s/558ce343893c8a88f4f8)
- **Paper Figures**: Included in `paper/figure/` directory
- **Analysis Scripts**: All processing scripts provided

### Not Included
- **Raw APKs**: Too large for distribution (~100GB+)
- **FlowDroid Raw Output**: Large intermediate files
- **AndroZoo Metadata**: Requires API access

## Usage Guidelines

1. **Citation**: Always cite both the paper and AndroZoo dataset
2. **API Limits**: Respect AndroZoo rate limits and terms of service
3. **Storage**: Ensure sufficient disk space for APK storage
4. **Processing Time**: Allow 2-3 weeks for complete replication
5. **Hardware**: Use systems with adequate RAM (8GB+ recommended)
